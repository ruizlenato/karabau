# Karabau

Karabau is an open-source Android client for [Karakeep](https://karakeep.app/), built from scratch using Jetpack Compose.

The name is a play on the original app’s name and a famous sound system culture from my home state of Pará, Brazil—specifically inspired by the “Aparelhagem Carabão”.

## Features

- Native Android app  
- Built with Jetpack Compose  
- Open-source alternative client for Karakeep
